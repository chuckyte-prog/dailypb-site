# Adding a Guest

Guests can be added to any DailyPB Session. Here are the steps to follow. Make sure to have the Guests cell phone number and email as you will need them to sign the Guest up.

[https://youtu.be/a5K1OzkiEB8](https://youtu.be/a5K1OzkiEB8)

1. Navigate to the Session and click on “View / Add Attendees.”
    
    ![image.png](/img/notion-help/adding-a-guest/image.png)
    
2. Scroll down to the section labeled “Manually Add Attendee” 
3. In the Search User section fill in either first and last name or email in the designated field. 
    
    ![image.png](/img/notion-help/adding-a-guest/image-1.png)
    
4. If there is no match in the DailyPB User database then a new section will appear below.
5. Make sure that in this bottom section, you have the correct info entered in the fields: first name, last name, and email.
6. You can now click the button “Manually Add Attendee.” 

![image.png](/img/notion-help/adding-a-guest/image-2.png)

- DailyPB will not allow a Guest to be created if a User exists with the same email. If a User exists with the same email the button to add the attendee will be disabled and the text “User Already Exists” will replace the text of the button.

![image.png](/img/notion-help/adding-a-guest/image-3.png)

- When that happens, simply enter the email in the Search Users section and click the button to “Add” the User to the Session.

![image.png](/img/notion-help/adding-a-guest/image-4.png)

# FAQ

***How do I add a Guest to the Waitlist if the Session is full?***

If the session is full, simply follow the sign-up instructions. DailyPB will automatically detect the full session and add the Guest to the Waitlist.

***What if I don't know the Guests email?***

It is important for you to enter the correct email and cell phone number when adding a Guest. Hosts need this information for messaging critical information.
