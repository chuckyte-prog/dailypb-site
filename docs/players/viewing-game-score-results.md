# Viewing Game Score Results

![image.png](/img/notion-help/viewing-game-score-results/image.png)

This is the Results Table

If your session has multiple brackets/boxes (ie: lower, middle, upper), select the “Filter” button, then use the dropdown box to select your desired bracket. 

![image.png](/img/notion-help/viewing-game-score-results/image-1.png)

Now you will only see games with that bracket or box. 

If you want to see the individual games, you can scroll down to view them. 

![chrome_ScllbzfnPv.gif](/img/notion-help/viewing-game-score-results/chrome-scllbzfnpv.gif)

Finally to only view the games of one player, simply click that player’s name or image in the Results Table. 

![image.png](/img/notion-help/viewing-game-score-results/image-2.png)

It will highlight, then the screen will scroll down to the induvial games, already filtered for that player.

![image.png](/img/notion-help/viewing-game-score-results/image-3.png)
