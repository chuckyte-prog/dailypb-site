# Signup Notifications

# Players

Players will get an signup notification when they sign up to a Session (waitlist or not) if the Session’s Host has a signup template enabled **or** the player has player notifications turned on (these are on by default - see image below) or if they were signed up by someone else. 

The only way a player will **not** get a signup notification is if the host doesn’t have a signup template enabled **and** the player has turned off their signup notifications **and** they signed up themselves. 

Players can find their notification settings inside their profile by clicking their name in the top right corner of the screen, then clicking on “Notifications.”

![Untitled](/img/notion-help/signup-notifications/untitled.png)

Players signed up by another player and guests (non-users) will always get a signup notification. 

This is the same whether the signup is for an attending spot or a waitlist spot.

# Hosts

All event hosts will get new player signup notifications if they have their host notifications turned on (this is on by default - see image below).
