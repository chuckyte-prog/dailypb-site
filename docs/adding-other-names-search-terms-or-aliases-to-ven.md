# Adding Other Names, Search Terms or Aliases to Venues to Help Players Search

1. Go to the Venue you want to edit
2. Click on Edit Venue
3. Find the section titled “Other Names”
4. You can now add and remove names for the Venue to help Users find the Venue more easily.
