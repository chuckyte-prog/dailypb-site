# Adding a User Manually

Users can be added to any DailyPB Session by another User. Here are the steps to follow. Make sure to have the Users cell phone number and email as you will need them to sign up.

1. Navigate to the Session and click on “View / Add Attendees.”
    
    ![image.png](/img/notion-help/adding-a-user-manually/image.png)
    
2. Scroll down to the section labeled “Manually Add Attendee” 
3. In the Search User section fill in either first and last name or email in the designated field. 
    
    ![image.png](/img/notion-help/adding-a-user-manually/35d4e773-fe81-44ea-bced-6e51d6346f37.png)
    
4. In the Search Results area you will see all Users that match. 
5. Find the one you wish to add and click the “Add” button next to their name.

![image.png](/img/notion-help/adding-a-user-manually/image.png)

# FAQ

***How do I add a User to the Waitlist if the Session is full?***

If the session is full, simply follow the sign-up instructions. DailyPB will automatically detect the full session and add the User to the Waitlist.

***Are first and last names case-sensitive?***

No, the first and last name fields are not case-sensitive. You can enter names in uppercase, lowercase, or a mix, and the system will still find the User. The email field is also case-insensitive.

***What if multiple Users match the same name?***

To ensure you're signing up the right person, search by their email address. An email search will always return a single, specific User.

***What if I don't know the Users email?***

Search by name. The search results will include the User's geographic location to help you identify the correct User.
