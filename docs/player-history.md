# Player History

Hosts can view the Signup history of any User who has previously signed up for a Session they hosted. 

![image.png](/img/notion-help/player-history/image.png)

# How to View Signup History

1. From the Attendees Popup, expand the Signup info by clicking the expand arrow on the Signup
2. Click on the text “View History”

![image.png](/img/notion-help/player-history/image-1.png)

You will now see a Popup that includes every Session in which the current User was the Session’s Event Host. 

# FAQ

***Why are the displayed Sessions limited to only those I hosted?***

This restriction is in place to protect User privacy.
