# How can I find the Pay or Cancel button

<aside>
💡

99.9% of the time users can't find the pay or cancel button when they load a Session — it is because they are NOT LOGGED IN

</aside>

## How a Non-Logged-In User Can Pay For a Session

Even if a user is not logged in they can actually still pay for a Session. All they need to do is navigate to the Session and click "View Attendees." 

![image.png](/img/notion-help/how-can-i-find-the-pay-or-cancel-button/image.png)

If they haven't paid and the Session is using automatic payments, they will see a Pay button next to their name that they can click.

![image.png](/img/notion-help/how-can-i-find-the-pay-or-cancel-button/image-1.png)
