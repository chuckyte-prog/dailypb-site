# Clone Event

Hosts can now clone Events using the Clone Event button located on the Edit Event screen. This will create a copy of the Event that is currently displayed on the screen. It will allow the Host to then add a new date and time to the cloned event for inclusion into the main schedule, Host Dashboard, Group events, etc. 

If the host fails to add a date and time, or cancels out of that screen, the event cloning process will need to be restarted.
