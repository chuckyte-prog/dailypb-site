# Updating All Future Session Times

Remember that an Event can have multiple Sessions. 

1. Make sure you’re logged in and then click “Global Event Tools” at the bottom of any Session that is part of the Event you’re trying to update.

![image.png](/img/notion-help/updating-all-future-session-times/e895a63c-c23e-4b4f-b486-2e7766376679.png)

1. Click “Edit Future Session Times”
    
    ![image.png](/img/notion-help/updating-all-future-session-times/image.png)
    
2. Update the start or end times for all future Sessions in 24 hour format (the text next to each box will show you the AM/PM equivalent)
3. Check the “Confirm” box 
4. Click the button “Update xx Sessions”
