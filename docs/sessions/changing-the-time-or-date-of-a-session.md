# Changing the Time or Date of a Session

If you need to change the time or date of a single Session, follow these instructions. 

<aside>
💡

Sometimes you will have a Session that gets cancelled due to rain or other reasons. Rather than making a new Session and moving everyone, consider simply changing the Session time and notifying all Players. (Shoutout to Katya for this idea) 

</aside>

1. Click on the Session you want to change
2. Click on “Change Time” under the Tools options (at the bottom).
3. You can now update the Session’s Start Time as well as End Time. 
4. If you have sign-ups attached to this Session, there will be a checkbox allowing them to be messaged about this change. 

The number in parenthesis indicates the number of Signups. If there is no waitlist the waitlist button will be disabled.
    
    ![image.png](/img/notion-help/changing-the-time-or-date-of-a-session/image.png)
    
5. When finished, click the “Change Session Time” button.

<aside>
😉

Don’t worry, any [Reminders](/help/sessions/2-24-hr-reminders) (2 and 24 hour) as well as Pre and Post [Templates](/help/sessions/email-templates) that maybe attached to your Session will be automatically rescheduled.

</aside>

## Changing Time / Date For Recurring Events

1.  Follow the instructions above.
2.  After you change one of the dates, you will see an option to change this Session and all following. 
    
    ![image.png](/img/notion-help/changing-the-time-or-date-of-a-session/image-1.png)
    
3.  You will now see the options for applying it to all following sessions. (If your reoccurring Event falls on more than one day, you will have the option to select which days you want the time change to apply to.)

![image.png](/img/notion-help/changing-the-time-or-date-of-a-session/image-2.png)

1.  Click Update Time. 

<aside>
⚠️

Note that notification emails will not be sent out to the Attendees or Wait-listers on any **future** Sessions that are being updated 

</aside>
