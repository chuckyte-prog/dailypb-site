# Pro Features for Hosts

Hosts that use Signups on their Sessions are given these features. 

This is regardless of whether they use [Manual](./manual-payments) (Venmo, Zelle, etc) or [Automatic Payments](./automatic-payments) (Stripe). 

# Events Management

1. Create and manage events/sessions with customizable details including [multiple hosts](./multiple-hosts-general-info)
2. Create private and public events
3. Set maximum attendance caps
4. [Manual](./manual-payments) or [Automatic Payment](./automatic-payments) tracking (automatic payment tracking only available for hosts using Stripe)
5. Support for multiple payment methods (Venmo, Zelle, Cash App, Paypal) 
6. Add up to 3 [Custom Fields](./custom-fields) for getting more info upon signup
7. [Printable Roster](./roster-of-signups) of attendees sorted by participant skill level
8. [Record Games](./viewing-game-score-results) and view stats
9. Create round robin or hybrid round robin ladder with up to 4 brackets and 96 players
10. [Event cloning](./clone-event) supported 

## Attendee Management

1. Gather [skill level](./skill-level-rating-requirements), cell phone number, and up to [3 custom fields](./custom-fields) of info on each player
2. Automatic host email with attendees, emails, and cell phone numbers 24 hours before session starts
3. Copy emails to clipboard, download all info to [spreadsheet](./spreadsheet-of-signups) (CSV)
4. Automatic [waitlist management](https://www.notion.so/4f4bcdce921942f58a397898b3602801?pvs=21) (auto-bumping) when sessions are full including text message notifications when a waitlister is bumped up.
5. [Manually add attendees](./adding-a-user-manually) to sessions
6. Mark attendees as paid/unpaid

## Communication Tools

1. Automatic player [reminders](./2-24-hr-reminders) 2 & 24 hours before session starts
2. Set up customized & automated [Email Templates](./email-templates) for 
    1. Immediate signup confirmation
    2. Pre-session communications
    3. Post-session communications
3. Receive notifications for new signups
4. Unlimited text-message all signups, paid participants, unpaid participants, or waitlisters (this will be paywalled at some point but for now it is free)

## Marketing Tools

1. [Interest Lists](./interest-lists)
    1. Players opt-in for your list to be notified of future sessions
    2. Ask up to 3 custom questions upon sign up
    3. Custom web page
    4. Export emails, names, or all to CSV
2. [Groups](./basic-group-info)
    1. All of your sessions in one place
    2. Custom URL for your group

## Financial Management

1. Track transactions in the Finance center
2. [Process payments through Stripe](./automatic-payments) or track payments [manually](./manual-payments) with Vennmo, Paypal, Cashapp, and Zelle integration
3. Automatic refunds for Stripe payees who cancel >24 hours ahead of your Session
4. Host can manually refund Stripe payees anytime
