# Multiple Hosts General Info

DailyPB allows multiple Hosts on Events. We will refer to the Host at the top of the List as the Main Host and all other Hosts as Sub-Hosts. 

# Public Visibility

Only the Main Host will show up as the Host when people view the Session and when they use the Message Host feature. There is no option for selecting a Sub-Host as the visible Host.

# Adding / Editing Events

Sub-Hosts can edit any Event in which they are listed as a Host. 

Assets (such as Interest Lists, Groups, and Email Templates) connected to an Event, even if they aren’t listed as Admins, will still be associated with the Event and visible on the Edit Event screen. However, they’ll still require Admin access to modify the details of these assets.

For instance, when the Main Host has an Interest List linked to an Event, and the Sub-Host lacks Admin access to this list, the Interest List will still appear as the linked Interest List to the Sub-Host. Nevertheless, the Sub-Host cannot modify the Interest List without Admin access.

# Automatic & Manual Payments

**Manual Payments Income** including the platform and account are all set in the Event Setting and are not limited to any particular Host account. Therefore, you can set a Sub-Host as the recipient of those manual payments.

**Manual Payments DailyPB Fee** - the DailyPB fee for Signups is tied to the Payment Method for the Main Host and cannot be changed to a Sub-Host. At the end of each month the Payment Method on File for the Main Host (at the time of the Transaction) will be assessed the total fees accrued for the month.

**Automatic Payments** always go to the Main Host’s Stripe Account and cannot be set up to be sent to anyone else. The main host can add, edit, modify, and delete their Automatic Payment settings via their Profile.
