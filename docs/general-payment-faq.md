# General Payment FAQ

***Does the host need to pay to sign up for their own event?***

Automatic Payments: No. Hosts can sign up without payment. The system automatically unchecks the "And Pay for Session" box when hosts register for their own Sessions. Other participants cannot uncheck this box. After signing up, hosts have the option to mark themselves as paid if desired.

Manual Payments: Yes. If the host signs up for the event, the standard DailyPB fee will be assessed to the Hosts. 

***What is the best link to use when asking unpaid Signups to pay?***

The best approach is to provide them with a link to the Session, which comes automatically in the DailyPB text messaging tool. When **logged in**, they'll see a "Pay Here" button directly on the Session popup.

Avoid sharing individual payment links since each URL is uniquely tied to a specific Signup and contains additional user information.

***Is there a way to set up an event as a series with a one-time payment?***

There is not currently a way to do that.

***Do hosts need to set up payment options for free events if they want to use Signups?***

No. 

***How do I know if I have Manual or Automatic Payments enabled?***

To check your payment settings, go to your Profile and click Monetization. There you'll see the status of both Automatic and Manual Payments. This section also lets you enable either payment type, adjust your settings, view your earnings, and manage your accounts.
