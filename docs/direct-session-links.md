# Direct Session Links

## Benefits of Direct Session Links

- Instant Access: Users can directly access the specific session without having to search or navigate through menus
- Easy Sharing: Share session details quickly with players via email, text, or social media
- Streamlined Registration: Players can immediately view session details and sign up from the direct link
- Perfect for Private Sessions: Direct links are the only way to access private sessions that aren't displayed publicly

## How to Get a Direct Session Link

There are two simple ways to get a direct link to a session:

### Method 1: Using Session Tools

- Open the session details popup
- Scroll down to find the Session Tools area
- Click the "Share Link" button
- The direct link will be automatically copied to your clipboard

![image.png](/img/notion-help/direct-session-links/image.png)

### Tips for Using Direct Links

- The link format will be: [https://dailypb.com?session=XXXXX](https://dailypb.com?session=XXXXX)
- Links are permanent and will work as long as the session exists
- Anyone with the link can view the session, even for private events
- Recipients can use the link to view session details, attendees, and sign up if spots are available
