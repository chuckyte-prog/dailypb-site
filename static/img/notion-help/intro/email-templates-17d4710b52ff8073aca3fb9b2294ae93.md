# Email Templates

<aside>
🔔

Create and modify Templates at [DailyPB.com/host](http://DailyPB.com/host) → Templates

</aside>

This tool lets hosts create personalized emails and schedule them to send at key times: upon signup, a custom number of hours before a session starts, or after it ends, simplifying administration.

# Create Email Template

1. Navigate to the Templates tab on your [Host](http://dailypb.com/host) page. You can click “Create New Template” and fill in the Template Name, Notes, and even add multiple Admins that will have full access to the Template.
2. Then you’ll want to write your Subject Line and message in the Edit Template area. Make sure to Save frequently.

![image.png](Email%20Templates/image.png)

1. You can preview your message anytime in the Preview Message area. This is especially helpful if you are using dynamic text blocks to customize your message as you can simulate what it will look like for a particular Session going to a particular player.

![image.png](Email%20Templates/image%201.png)

1. Make sure to Save your Template
2. At the bottom of the Preview Message area you can send a test email.

## Dynamic Text Blocks

![image.png](Email%20Templates/image%202.png)

Dynamic text blocks allow you to personalize your emails for the Signup and the Session. Simply enter the text block (ex: [name]) into your Template to customize and see what it looks like in the Preview Message area.

# Link Email Template to An Event

You can link an Email Template to any Event you host. Here are the steps to do this. 

1. At the bottom of a Session Popup, click on “Global Event Tools.”
2. Click on “Edit Event.”
3. Navigate to the 5th page of the Edit Event menu to find the Template settings
    
    ![image.png](Email%20Templates/image%203.png)
    
4. Use the dropdown on any section and select the Template you want to use

![image.png](Email%20Templates/image%204.png)

1. If you are setting the “X hours before a Session starts” or “X hours after a Session ends” make sure to end the appropriate number of hours.
    
    <aside>
    💡
    
    You will know a Template is Active if you see the “Active” checkmark within that Template area.
    
    ![image.png](Email%20Templates/image%205.png)
    
    </aside>
    
2. Click the “Save & Close” button
