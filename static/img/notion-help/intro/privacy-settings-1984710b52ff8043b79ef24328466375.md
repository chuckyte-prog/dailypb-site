# Privacy Options

**Public** - anyone can view Sessions

**Group** - only Group members (of a linked Group) can view Sessions. 

**Private** - only those with direct link access can view Sessions

# Session Visibility

|  | Public Schedule | Group Page | Direct URL |  |
| --- | --- | --- | --- | --- |
| Public | Yes | Yes | Yes |  |
| Group | No | Only Group Members | Yes |  |
| Private | No | No | Yes |  |

# How to Change the Privacy Settings

1. Click on “Edit Session” at the bottom of any Session
2. Click on “Audience & Display” 
3. Find the section “Who can see it?”
4. Click on the appropriate option.
5. Click “Save Changes” to save your settings

If you have more questions about why your Session may not be showing up, check out this [page](Why%20isn't%20my%20Session%20showing%20up%20on%20the%20main%20schedu%202c44710b52ff8050b91de177c3643020.md)
