# Why isn't my Session showing up on the main schedule?

This is nearly always due to the visibility setting in a Session. Here are the main things to check:

1. Visibility = Public
2. Search Area (via User’s Location Preferences) match Session’s Venue

## Visibilty = Public

![image.png](Why%20isn't%20my%20Session%20showing%20up%20on%20the%20main%20schedu/image.png)

To view or edit the Session visibility: 

1. Click on "Edit Session" at the bottom of that Session
2. Click on “Audience and Display”
3.  Under the section "Who can see it?" you will see the current visibility settings. 

## Matching Location Preferences

![image.png](Why%20isn't%20my%20Session%20showing%20up%20on%20the%20main%20schedu/image%201.png)

Every User has a location and a search radius, which you can think of as a circle on a map with the center dot at the user's location. 

Any Sessions which have Venues that fall within the search radius will be shown. 

Any Sessions with Venues that are outside of the radius will not be shown. 

![search radius.png](Why%20isn't%20my%20Session%20showing%20up%20on%20the%20main%20schedu/search_radius.png)
