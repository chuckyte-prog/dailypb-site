# Can I make an Event private or in a “draft” mode?

Yes, if you make the visibility of the Event: Private and make sure not to link any Group to the Event then only the Event Host(s) and the Venue Admin will be able to see your Event. 

![image.png](Can%20I%20make%20an%20Event%20private%20or%20in%20a%20%E2%80%9Cdraft%E2%80%9D%20mode/image.png)

The Venue Admin is the Admin of the Venue you have chosen for your Event.
