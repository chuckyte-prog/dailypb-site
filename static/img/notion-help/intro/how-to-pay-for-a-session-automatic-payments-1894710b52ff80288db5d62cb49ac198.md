# How to Pay for a Session (Automatic Payments)

This help article is for Sessions with Automatic Payments.  

# How to Pay for Yourself

1. Make sure you are logged in.
2. Click on the Session you are signed up for. 
3. Click on the “Pay Here” button in the RSVP info section.
4. You will be sent to a Stripe Checkout Session where you can pay
5. After paying you will be redirected back to DailyPB

![image.png](How%20to%20Pay%20for%20a%20Session%20(Automatic%20Payments)/image.png)

# How to Pay for Someone Else or 
How to Pay if You are Not Logged In

1. Click on the Session
2. Click “View / Add Attendees” to view the Attendees Popup
3. Find the person for whom you wish to pay. 
4. Click the “Pay” button next to their name
5. You will be sent to a Stripe Checkout Session where you can pay

![chrome_NU3pHNOcIP.gif](How%20to%20Pay%20for%20a%20Session%20(Automatic%20Payments)/chrome_NU3pHNOcIP.gif)

If you are not seeing any buttons with the words “Pay” on them, your host might be using Manual Payments. [How to Pay for a Session with Manual Tracking](How%20to%20Pay%20for%20a%20Session%20(Manual%20Payments)%201894710b52ff80d19725c21d442ac63a.md)

# FAQ

***How can I know if the Session is using Automatic or Manual Payments?***

You will know if the Session uses Automatic Payments by clicking the info icon next to the Price of the Session and if it shows a Stripe Fee, then the Session uses Automatic Payments. If not, it uses Manual Payments.
