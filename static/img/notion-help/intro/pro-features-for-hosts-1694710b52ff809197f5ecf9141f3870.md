# Pro Features for Hosts

Hosts that use Signups on their Sessions are given these features. 

This is regardless of whether they use [Manual](Manual%20Payments%201534710b52ff80549bcce132eb21089e.md) (Venmo, Zelle, etc) or [Automatic Payments](Automatic%20Payments%201534710b52ff80a1a3c2cdd2a27d8aaf.md) (Stripe). 

# Events Management

1. Create and manage events/sessions with customizable details including [multiple hosts](Multiple%20Hosts%20General%20Info%201764710b52ff8011a360d0b376eddaea.md)
2. Create private and public events
3. Set maximum attendance caps
4. [Manual](Manual%20Payments%201534710b52ff80549bcce132eb21089e.md) or [Automatic Payment](Automatic%20Payments%201534710b52ff80a1a3c2cdd2a27d8aaf.md) tracking (automatic payment tracking only available for hosts using Stripe)
5. Support for multiple payment methods (Venmo, Zelle, Cash App, Paypal) 
6. Add up to 3 [Custom Fields](Custom%20Fields%2017d4710b52ff80799dabc7524ad881a2.md) for getting more info upon signup
7. [Printable Roster](Roster%20of%20Signups%2017d4710b52ff80919687d0772208cac4.md) of attendees sorted by participant skill level
8. [Record Games](Viewing%20Game%20Score%20Results%200a2116d6971747fba319765bb709df20.md) and view stats
9. Create round robin or hybrid round robin ladder with up to 4 brackets and 96 players
10. [Event cloning](Clone%20Event%20f4e2811f4bc44778a6eb06a0d297cfcc.md) supported 

## Attendee Management

1. Gather [skill level](Skill%20Level%20Rating%20Requirements%2017d4710b52ff804d8256da743b58abe2.md), cell phone number, and up to [3 custom fields](Custom%20Fields%2017d4710b52ff80799dabc7524ad881a2.md) of info on each player
2. Automatic host email with attendees, emails, and cell phone numbers 24 hours before session starts
3. Copy emails to clipboard, download all info to [spreadsheet](Spreadsheet%20of%20Signups%2017d4710b52ff80d983f1c5c743fdbbac.md) (CSV)
4. Automatic [waitlist management](https://www.notion.so/4f4bcdce921942f58a397898b3602801?pvs=21) (auto-bumping) when sessions are full including text message notifications when a waitlister is bumped up.
5. [Manually add attendees](Adding%20a%20User%20Manually%2017d4710b52ff809abfe2f5d49f962611.md) to sessions
6. Mark attendees as paid/unpaid

## Communication Tools

1. Automatic player [reminders](2%20&%2024%20Hr%20Reminders%2017c4710b52ff8098860bd2402894f811.md) 2 & 24 hours before session starts
2. Set up customized & automated [Email Templates](Email%20Templates%2017d4710b52ff8073aca3fb9b2294ae93.md) for 
    1. Immediate signup confirmation
    2. Pre-session communications
    3. Post-session communications
3. Receive notifications for new signups
4. Unlimited text-message all signups, paid participants, unpaid participants, or waitlisters (this will be paywalled at some point but for now it is free)

## Marketing Tools

1. [Interest Lists](Interest%20Lists%201304710b52ff8053ba51cbd71c12fdcf.md)
    1. Players opt-in for your list to be notified of future sessions
    2. Ask up to 3 custom questions upon sign up
    3. Custom web page
    4. Export emails, names, or all to CSV
2. [Groups](Basic%20Group%20Info%2017d4710b52ff80d09d23fd89ddf67671.md)
    1. All of your sessions in one place
    2. Custom URL for your group

## Financial Management

1. Track transactions in the Finance center
2. [Process payments through Stripe](Automatic%20Payments%201534710b52ff80a1a3c2cdd2a27d8aaf.md) or track payments [manually](Manual%20Payments%201534710b52ff80549bcce132eb21089e.md) with Vennmo, Paypal, Cashapp, and Zelle integration
3. Automatic refunds for Stripe payees who cancel >24 hours ahead of your Session
4. Host can manually refund Stripe payees anytime
