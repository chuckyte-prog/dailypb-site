# How to Create a Direct Payment Link

When using Automatic Payments, anyone can create a direct payment link by clicking “Pay” next to a Player’s name, waiting for the Stripe Checkout Session window to load, then copying the URL from that window. 

<aside>
🚧

Direct payment links are only active for 24 hours.

</aside>

# How to Create a Direct Payment Link

1. Open the **Session Details**.
2. Click **View/Add Attendees** to open the **Attendees Popup**.
3. Locate the attendee for whom you want to generate a direct payment link and click the **Pay** button next to their name.
*Note:*  Payment links are specific to each signup, so ensure you select the correct attendee.
4. Once the Stripe Checkout session loads, copy the URL from your browser.

You can now share this link with the attendee, and it will open the Stripe Checkout session for them. Remember the link will expire in 24 hours.
