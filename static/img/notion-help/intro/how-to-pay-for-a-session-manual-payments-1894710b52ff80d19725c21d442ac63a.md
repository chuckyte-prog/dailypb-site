# How to Pay for a Session (Manual Payments)

If your host has enabled Manual Payments on the Session, there will be links to the Payment Method they prefer (ex: Venmo, Zelle, Cashme, Paypal).

# Paying When You Signup

It is best to pay the Host for the Session immediately after you sign up for the Session. After confirming your participation, a screen will appear with information on how to pay the host. Make sure to pay at this time or message your Host if you are having difficulties.

# Paying without a Signup or
Paying after you Signup

On the Session details screen you will see a section titled “Payment Methods” where you will find information on how to pay the Host. 

![image.png](How%20to%20Pay%20for%20a%20Session%20(Manual%20Payments)/image.png)

If you are not seeing the same screens and info as described here, it is possible the Session is either using [Automatic Payments](How%20to%20Pay%20for%20a%20Session%20(Automatic%20Payments)%201894710b52ff80288db5d62cb49ac198.md) or there is no fee. 

# FAQ

***How can I know if the Session is using Automatic or Manual Payments?***

You will know if the Session uses Automatic Payments by clicking the info icon next to the Price of the Session and if it shows a Stripe Fee, then the Session uses Automatic Payments. If not, it uses Manual Payments.

![image.png](How%20to%20Pay%20for%20a%20Session%20(Manual%20Payments)/image%201.png)

***I’ve signed up, but I don’t see any buttons to Pay anywhere?***

Make sure you are logged in. You will know if you are logged in if your name appears in the upper right-hand corner of your screen. 

![image.png](How%20to%20Pay%20for%20a%20Session%20(Manual%20Payments)/image%202.png)

***Okay, I’ve signed up, and I’m logged in, but I still don’t see buttons to Pay when I open the Session.***

### Confirm the Account Linked to Your Signup

1. **Check Your Status in the Session:**
    - If you are logged into the correct account and signed up for the session, you will see **“Attending Event”** and **“Cancel My Spot”** displayed in the session details.
    - If you do not see this information, you may not be logged into the account linked to the signup.
    

![This info will only show if you are logged in and Attending an Event.](How%20to%20Pay%20for%20a%20Session%20(Manual%20Payments)/image%203.png)

This info will only show if you are logged in and Attending an Event.

1. **Verify Your Account:**
    - It’s possible you have multiple accounts in DailyPB. To determine which account is linked to your signup:
        - Locate the signup confirmation email sent by DailyPB.
        - Check the email address it was sent to; this corresponds to the account used for the signup.
2. **Contact the Host if Needed:**
    - If you’re unable to find the confirmation email, contact the host and ask which email address was used to sign you up for the session.

This process ensures you are logged into the correct account to access your session details.

### Confirm the Session is using Automatic Payments

See FAQ section above. If the Session is not using Automatic Payments then you won’t see any “Pay” or “Pay Here” buttons.
